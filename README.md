## Setup
1. Clone or download the project files.
2. Open `index.html` in a web browser.
3. Replace placeholder images in the `images/` folder with your actual portfolio images:
   - `profile.jpg`: Your profile picture (already added)
   - `project1.svg`, `project2.svg`, `project3.svg`: Your project images

## Customization
- Edit `index.html` to update content, projects, and contact information.
- Modify `css/style.css` for custom styling.
- Add functionality in `js/script.js`.
